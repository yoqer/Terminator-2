# Terminator-2

Terminator 2. Segunda versión del Terminator 1, que eleva desde Kokoro, con funciones más próximas del Terminator I y el del nuevo, más desligado [Terminator II](https://github.com/yoqer/Terminator-II-).

Con multiples provedores de voz, y modelos; desde uso local a VPS o web Hosting. Incluso uso por API.
